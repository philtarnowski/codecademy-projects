# Website Design System

A front-end reference project built as part of the Codecademy web development curriculum.

## Overview

This project documents the core visual language for a website, including a color palette, typography scale, and reusable UI components. It serves as a living style guide for building consistent interfaces.

## Files

- `index.html` — Page structure and content
- `styles.css` — Styles for colors, typography, and components

## Skills Practiced

- Building a color palette with CSS background colors and hex values
- Displaying font families and type styles
- Laying out sections with CSS
- Structuring a multi-section reference page with HTML

## How to View

Open `index.html` in any browser — no build step or dependencies required.
