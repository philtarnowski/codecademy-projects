HTML Tables Cheatsheet

Codecademy Project

The goal of this Codecademy project is to set up an index file, connect a stylesheet, and style a few elements in a basic table.
Requirements are also to include a basic reference guide for some key rules and tags used in HTML tables.

Rule Category Key Tags / Attributes Notes
Structure <table>, <tr>, <th>, <td> Nesting order must be respected
Organization <thead>, <tbody>, <tfoot> Groups rows by purpose
Spanning colspan, rowspan Merges cells across columns or rows
Styling border, border-collapse, padding Applied via CSS
