# HTML Tables Cheatsheet

A simple reference project built as part of the Codecademy web development curriculum.

## Overview

This project demonstrates how to structure and style HTML tables. It includes a cheatsheet covering the core tags, attributes, and CSS properties used when working with tables in HTML.


## Files

- `index.html` — Page structure and table content
- `styles.css` — Styles for headings, table headers, cells, and body text

## Skills Practiced

- Structuring HTML tables with `<thead>` and `<tbody>`
- Connecting an external stylesheet
- Applying CSS to table elements (`th`, `td`) including borders and padding
- Basic typographic styling with `font-family`, `color`, and `font-size`

## How to View

Open `index.html` in any browser — no build step or dependencies required.
