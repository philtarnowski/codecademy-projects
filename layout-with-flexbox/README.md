# Layout with Flexbox

A single-page website built as part of the Codecademy web development curriculum.

## Overview

This project involved recreating a fictional tea shop website from a provided design. The design defined exact measurements, colors, and layout structure, and the goal was to match it as closely as possible using only HTML and CSS.

## Files

- `index.html` — Page structure and content
- `styles.css` — Styles for layout, typography, and background images

## Skills Practiced

- Building a fixed navigation bar with a logo and inline links
- Laying out sections with Flexbox including row and column directions
- Using `justify-content` and `align-items` to center content horizontally and vertically
- Applying background images with `cover` sizing and centered positioning
- Offsetting content to account for a fixed header with `padding-top`

## How to View

Open `index.html` in any browser — no build step or dependencies required.
